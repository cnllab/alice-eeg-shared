## Alice EEG 

Preprocessing and analysis for EEG data collected while participants listend to Chapter 1 of *Alice in Wonderland*

### Getting started

- Download the data
- Download prePROCessing parameters
- install fieldtrip

### Background

- Data collected between X and Y
